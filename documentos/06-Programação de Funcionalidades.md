## Tela de Login 

<h4>Responsavel: Kaique Silveira Sampaio</h4>

Descrição

A tela de login constitui o ponto de entrada do sistema, sendo responsável pela autenticação dos usuários e pelo controle de acesso às funcionalidades da aplicação. Por meio da validação das credenciais informadas, o sistema identifica o perfil do usuário e disponibiliza apenas os recursos compatíveis com suas permissões.

Essa funcionalidade contribui para a segurança das informações armazenadas, impedindo acessos não autorizados e garantindo que cada colaborador visualize apenas as ferramentas necessárias para a execução de suas atividades.

<h3>Instruções de Acesso</h3>

Ao iniciar a aplicação, o usuário deve informar seu nome de usuário e senha nos campos correspondentes. Após a validação dos dados, o sistema direciona automaticamente o usuário para a área apropriada de acordo com seu perfil de acesso.

## Tela de Cadastro de Usuários

<h4>Responsavel: Kaique Silveira Sampaio</h4>

Descrição

A funcionalidade de cadastro de usuários permite a inclusão de novos colaboradores no sistema, garantindo que cada membro da organização possua uma conta individual para acesso à plataforma. Durante o cadastro, são registradas informações essenciais para identificação do usuário e definição de suas permissões de utilização.

Essa funcionalidade é fundamental para a administração do sistema, uma vez que possibilita o gerenciamento dos perfis de acesso e contribui para a rastreabilidade das operações realizadas pelos usuários.

<h3>Instruções de Acesso</h3>

O administrador deve acessar a opção "Cadastrar Usuários" no menu lateral, preencher os dados solicitados e selecionar o perfil adequado para o novo usuário antes de concluir o cadastro.

## Painel de Controle (Dashboard)

<h4>Responsavel: Kaique Silveira Sampaio</h4>

Descrição

O painel de controle concentra as principais informações operacionais do sistema em uma única interface, fornecendo uma visão geral do ambiente produtivo.
